# ENPER

ENPER is a website for English teachers to easily look up study guides and book summaries. The contents of the books are organized into chapters and pages, allowing teachers to quickly read and understand the book's contents.

## Features

- View book summaries and study guides.
- Add contents such as chapter questions.
- Manage and organize student schedules.
- Create a list of worksheets for students

## Technologies Used

- HTML, CSS, JavaScript, jQuery, Ajax, Thymeleaf
- MDB - Material Design for Bootstrap
- Java, Spring Boot 3.1.2, Spring Security
- Gradle, JDBC, MySQL, Lombok, JPA, Validator
- Ubuntu (VPS)
- SummerNote API (WYSIWYG Editor)
